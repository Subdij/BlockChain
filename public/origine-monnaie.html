D'où vient l'argent dans notre système blockchain ?

Une question très importante se pose : comment est-il possible de gagner de l'argent en minant alors qu'on commence avec 0€ ? D'où vient exactement cet argent ?

Point clé : Dans une blockchain, l'argent n'est pas simplement transféré d'une personne à une autre - il est également créé par le système lui-même selon des règles prédéfinies.

La création monétaire par minage

Dans notre système blockchain (comme dans Bitcoin), l'argent est créé par le protocole lui-même selon des règles fixes et transparentes. Voici comment cela fonctionne :

  1. Émission programmée : À chaque bloc miné avec succès, le protocole génère automatiquement une récompense (50€ dans notre cas).
  2. Transaction "coinbase" : Cette récompense est créée à partir de rien et accordée au mineur via une transaction spéciale appelée "coinbase".
  3. Ajout à la masse monétaire totale : Ces nouvelles unités sont ajoutées au montant total d'argent en circulation.

Explication technique

Quand un mineur mine un bloc, la première transaction du bloc (transaction "coinbase") est spéciale :

// Transaction coinbase générée automatiquement
{
  from: "COINBASE", // Pas d'expéditeur réel
  to: "[adresse du mineur]",
  amount: "50", // Récompense fixe
  fee: "0",
  coinbase: true
}

Cette transaction est unique car :

  • Elle n'a pas d'expéditeur réel (marqué "COINBASE")
  • Les fonds sont créés directement par le protocole
  • Elle est toujours placée en premier dans le bloc

Comparaison avec le système monétaire traditionnel

Système traditionnel (euro, dollar...) Système blockchain
Création monétaire par les banques centrales Création monétaire par le protocole selon des règles fixes
Décisions discrétionnaires (par des humains) Règles algorithmiques (programmées à l'avance)
Quantité potentiellement illimitée Quantité souvent limitée (ex: Bitcoin limité à 21 millions)
Création par émission de dette Création par récompense de travail (minage)

Pourquoi ce système ?

Ce mécanisme de création monétaire par minage offre plusieurs avantages :

  • Distribution équitable : L'argent est distribué à ceux qui contribuent à la sécurité du réseau
  • Incitation : Encourage les mineurs à valider les transactions et à sécuriser le réseau
  • Prévisibilité : Les règles d'émission sont connues à l'avance, contrairement aux monnaies traditionnelles
  • Décentralisation : Pas besoin d'une autorité centrale pour émettre la monnaie

Implémentation dans notre simulation

Dans notre code, la création de la récompense de bloc est implémentée comme suit :

// Créer la transaction coinbase (récompense du mineur)
const coinbaseReward = BLOCK_REWARD + totalFees;
const coinbaseTx = {
  from: "COINBASE",
  to: minerPublicKey,
  amount: coinbaseReward.toString(),
  fee: "0",
  coinbase: true
};

// Ajouter la transaction coinbase en première position
transactions.unshift(coinbaseTx);

Puis, une fois le bloc miné, le solde du wallet du mineur est mis à jour pour refléter cette nouvelle création monétaire :

// Mettre à jour le wallet du mineur avec la récompense
function updateMinerWallet(reward) {
  const minerWallet = availableWallets[selectedWalletIndex];
  minerWallet.balance = (parseFloat(minerWallet.balance) + parseFloat(reward)).toFixed(2);
  // Mettre à jour localStorage
  localStorage.setItem('blockchainWallets', JSON.stringify(availableWallets));
}

Conclusion

En résumé, l'argent que vous gagnez en minant ne provient pas d'autres utilisateurs mais est créé par le protocole lui-même. C'est un mécanisme fondamental qui permet à une blockchain de fonctionner sans autorité centrale tout en distribuant équitablement de nouvelles unités monétaires.

Dans notre simulation, vous pouvez observer ce mécanisme de création monétaire en action :

  2. Minez un bloc sur la page Mining ou Transaction
